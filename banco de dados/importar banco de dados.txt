Para importar um banco de dados no MySQL   :

    1º inicie o banco de dados
	2º selecione o banco de dados:
	3º clique na aba server
		3.1 selecione se vai importar em um banco de dados já existente ou não
	4º clique em ‘data import’
	5º seleciona a pasta do dump
	6º clique em start import
	7º coloque a senha do servidor
